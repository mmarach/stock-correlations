About Stock Correlations App

Stock Correlations is a simple tool which calculates the correlations between selected stocks.

Computing Correlations

To compute correlations, start by entering the stock tickers of interest using the Add Ticker field. Each ticker must be valid and available in the Yahoo Finance database. After adding the tickers, specify the date range for the calculation, then click Submit to generate a correlation matrix displaying the computed values.

Additional Calculation Options

You can choose whether to calculate correlations using daily returns or raw prices. By default, daily returns are used, but you can switch to raw prices by unchecking the Use Price Returns option.

Additionally, you can decide whether to adjust the underlying price data for applicable corporate actions such as stock splits and dividend distributions. To use unadjusted price data, uncheck the Adjust for Corporate Actions option.

Yahoo Finance API

Stock Correlations utilises Yahoo's publicly available APIs to obtain historical price data from Yahoo Finance using which the correlations are computed.

For more information on Yahoo's APIs, see their official documentation.
